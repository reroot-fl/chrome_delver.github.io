# CHROME / DELVER

> **クローム・デルヴァー** — 鉄屑の底へ潜り、機械都市の心臓部からコアを奪い、崩れゆく廃墟を生きて帰れ。

ブラウザだけで動く、ターン制ローグライク。レベルアップは無い。**お前の装備が、お前のすべてだ。**

## 概要

`CHROME DELVER` は、依存ライブラリ無しの単一 HTML ファイルで完結するサイバーパンク風ローグライクです。プレイヤーはスクラップの海に潜る自律機械（コア）を操作し、敵を破壊して奪ったパーツで自分自身を組み替えながら最深部 **B10** を目指します。

このゲームには経験値もレベルもありません。強さはすべて装備（武器・装甲・ジェネレータ）と、現地で拾うアップグレードから生まれます。

## ゲームの目的

1. **潜行（ダイブ）**: B1 から階段を下り、B10 まで降りる。
2. **回収**: B10 の最深部に眠る **コア 💠** を回収する。
3. **生還（リターン）**: コアを取った瞬間に施設が崩壊を始める。階段は上昇用に切り替わり、**追跡者 👁️** が出現。崩れ続けるフロアを抜けて **B1 の脱出口 🚪** までたどり着けばクリア。

## 特徴

- **単一ファイル / ゼロ依存** — `chrome_delver.html` を開くだけ。ビルド不要。
- **二相構造** — 「下りの潜行」と「上りの脱出」で緊張感がガラリと変わる。
- **装備がすべて** — レベル無し。武器・装甲・ジェネレータの組み合わせがビルド。
- **リスクとリターン** — メモリーチップ、武器マルウェア、ターミナルのセキュリティアラートなど、博打要素が多数。
- **CRT風の演出** — 走査線・グリッチ・フローティングダメージなど。

## 操作方法

PCゲーム標準の WASD 配置に最適化されています。`WASD` から指を離さず主要アクションを押せます。

| キー | 操作 |
|---|---|
| `W` `A` `S` `D` / 矢印 / `H` `J` `K` `L` | 移動・攻撃（敵のいるマスへ進むと攻撃） |
| `E` または `F` | 調べる（拾う／使用／階段の昇降／ターミナル操作） |
| `Q` | 分解（足元・隣接アイテムを資源へ） |
| `R` | 修復（電力10消費でHP5回復） |
| `C` | オーバークロック |
| `Space` | 待機（1ターン経過） |
| `Esc` | UI（ターミナルパネル等）を閉じる |

モバイルでは画面下部の D-Pad とアクションボタンで同等の操作ができます。

## リソース管理

- **HP** — コアの耐久。0 でゲームオーバー。装甲の換装で最大値が変わる（換装時に全回復）。
- **電力 (PW)** — **5歩ごとに -1**。0 になると歩くたびに HP が削れる。バッテリーやジェネレータ換装で回復・拡張。
- **ATK** — 武器の攻撃力 + メモリーチップによる基礎ATK。攻撃・被弾には **10% の確率で MISS / 回避** がある。

## 主要システム

### ⚡ オーバークロック
電力5を消費して **3ターンの強化**（ATK×2／被ダメージ半減）。終了後は **3ターンのオーバーヒート**（ATK半減＋毎ターン HP ダメージ）という反動が来る。ここぞの場面で使う諸刃の剣。

### 🔨 分解（スクラップ）
拾ったパーツを装備せず資源に変換する。武器→次の一撃を強化、装甲→HP回復、バッテリー/ジェネレータ→電力回復。装備するか分解するかの判断がビルドの肝。

### 💾 メモリーチップ / マルウェア
チップは基礎ステータスを永続的に底上げするが、低確率で逆効果のマルウェアを引く。さらに武器を換装した際も一定確率で **武器がシステムロック**され、アンチウイルス 💿 を拾うまで換装できなくなる。

### 🖥️ ターミナル
旧世代端末にアクセスすると以下から1つ選択できる。ただし **30% の確率でセキュリティアラート**が発動し、敵部隊が緊急展開する。
- 📡 **フロア・スキャン** — フロア全域をマッピング
- ⚡ **電力抽出** — 電力 +15
- 📦 **パーツ・サルベージ** — ランダムな装備をドロップ

### 🏚️ 帰還フェーズ（崩壊）
コアを回収すると世界が一変する。
- 階段の挙動が **下降 → 上昇** に切り替わる。
- 上の階へ戻るたびに **マップが新規生成**される（施設の崩壊）。
- 各フロアに **追跡者 👁️** が出現し、立ち止まると追いつかれる。

## 技術仕様

- **言語**: 素の HTML / CSS / JavaScript（フレームワーク無し）
